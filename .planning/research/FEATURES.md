# Features Research: Markdown Rendering

## Table Stakes (Must Have)

- Markdown to HTML conversion with GFM support
- GitHub-style visual styling
- Code block syntax highlighting
- Language label display on code blocks
- Copy-to-clipboard functionality

## Differentiators (Competitive Advantage)

- Theme switching (light/dark)
- Smooth scroll synchronization (optional)
- Custom code block headers
- Line numbers in code blocks

## Anti-Features (Deliberately Not Building)

- Real-time preview editing
- Markdown editing with toolbar
- Export to PDF/DOCX
- Collaborative editing
- Version history

## Feature Complexity

| Feature | Complexity | Dependencies |
|---------|------------|--------------|
| marked integration | Low | None |
| github-markdown-css | Low | None |
| highlight.js | Medium | Language detection |
| Copy button | Low | Clipboard API |
| Theme toggle | Low | CSS variables |
| Dual scroll panels | Low | CSS layout |
