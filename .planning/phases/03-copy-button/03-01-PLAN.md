  <name>Add copy button to code block renderer</name>
  <files>projects/ngx-mkd/src/lib/ngx-mkd.ts</files>
  <action>
    Modify the renderer.code function in ngx-mkd.ts to include a copy button in the code block wrapper:

    1. Add a button element with class "copy-button" and text "Copy" to the HTML returned by renderer.code
    2. Position the button in the top-right corner of .code-block-wrapper (same row as .code-lang)
    3. The button should have a data attribute with the raw code content for copying

    Existing code structure to modify:
    ```typescript
    renderer.code = function({ text, lang }: { text: string; lang?: string }) {
      const language = lang || 'text';
      // ... existing highlighting logic ...
      return `<div class="code-block-wrapper">
        <span class="code-lang">${language}</span>
        <pre><code class="hljs language-${language}">${highlighted}</code></pre>
      </div>`;
    };
    ```

    New structure:
    ```typescript
    return `<div class="code-block-wrapper" data-code="${encodeURIComponent(text)}">
      <span class="code-lang">${language}</span>
      <button class="copy-button" data-copied="Copy" data-original="Copy">Copy</button>
      <pre><code class="hljs language-${language}">${highlighted}</code></pre>
    </div>`;
    ```
  </action>
  <verify>

  </verify>
